# RSA 暗号について
## 秘密鍵と公開鍵の普通の作り方
1. p と q をランダムに選ぶ(ただし、p と q はともに素数)
2. N = pq とすると、φ(N) = (p - 1)(q - 1)
3. GCD(e, φ(N)) = 1 かつ 0 < e < N を満たす e を選ぶ
4. ed mod φ(N) = 1 かつ 0 < e < N を満たす d を選ぶ
5. N, e を公開鍵とし、d を秘密鍵とする

# N　を因数分解して頑張る
`./msieve -e -v -q (N の数)`  で因数分解して p と q を求める

# e が極端に小さいとき (e = 3)
